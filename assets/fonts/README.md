# Polices RIPA

Deux dossiers à utiliser :

- **microgramma/** – Priorité : **Microgramma Bold Extended** (`MicrogrammaDBoldExtended.ttf`) pour titres et logo.
- **roboto/** – Priorité : **Roboto Regular** (`Roboto-Regular.ttf`) pour le corps de texte.

Chargement dans l'app via `expo-font` (voir `App.js` et `src/constants/theme.js`).
